AJAX란?(에이잭스)

Asynchronous JavaScript and XML

XML으로 페이지가 로딩 된 후 데이터를 요청하여 받을 수 있는 비동기식 자바스크립트. 즉 전체 페이지를 새로 고침하지 않고 필요 데이터만 교환

http 에서 일어나는 인터페이스

JSONS이란?

JavaScript Object Notation

순수 정보만 보내기 위해 만들어진 경량화된 포맷

자바스크립트에서만 쓰는 양식이 아님. 다른 언어에서도 굉장히 많이 사용

https://www.oracle.com/kr/database/what-is-json/ mdn

데이터 유형 중 undefined 는 안들어감

반드시 큰 따옴표 사용

수정전
{ 'title':'bg1', "description" :"배경이미지 1입니다", "id" : 1, "type" : undefined }

수정후
{ "title":"bg1", "description":"배경이미지 1입니다", "id":1, "type":"undefined" }

undefined를 문자열로 인식하기 때문에 큰 따옴표로 변환해줌
